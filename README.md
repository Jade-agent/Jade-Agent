# Jade-Agent

参考书籍：基于JADE平台的多Agent系统开发技术(作者：于卫红、陈燕)  

## 项目开发环境说明  

> Jdk 1.7  
Jade 4.4.0  
编译器 Eclipse  

环境配置参考`Agent01Agent`里面的说明
